# Quiz
O Quiz é uma plataforma interativa desenvolvida para o Centro de Estudos Cognita, com o objetivo de tornar a aprendizagem mais dinâmica, divertida e eficaz.

Através de quizzes educativos, os alunos podem testar os seus conhecimentos, reforçar conteúdos escolares e acompanhar a sua evolução de forma prática e intuitiva.

Link: https://cognitaestudo.github.io/Quiz/
